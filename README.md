# Molecular-Property-Prediction-using-Explainable-Graph-Neural-Networks
Explainable Graph Neural Network framework for molecular property prediction on the QM9 dataset using Graph Isomorphism Networks (GIN). Integrates SHAP and LIME to provide global and local explanations of model behavior while predicting HOMO, LUMO, Band Gap, and Dipole Moment properties.

# Explainable Graph Neural Networks for Molecular Property Prediction

## Project Overview

This project explores molecular property prediction using Graph Isomorphism Networks (GIN) on the QM9 dataset. In addition to predictive modeling, the project focuses on explainability by integrating SHAP and LIME to interpret the learned graph representations generated by the network.

The framework predicts four important quantum chemical properties:

* HOMO Energy
* LUMO Energy
* HOMO–LUMO Band Gap
* Dipole Moment

Unlike traditional machine learning approaches that rely on handcrafted descriptors, the model learns molecular representations directly from graph-structured molecular data using PyTorch Geometric.

## Model Architecture

The model is based on the Graph Isomorphism Network (GIN) architecture.

### Network Configuration

* 3 GIN Convolution Layers
* Hidden Dimension: 64
* Batch Normalization
* ReLU Activation
* Global Add Pooling
* Fully Connected Regression Head
* Adam Optimizer
* ReduceLROnPlateau Scheduler

## Explainability Framework

Graph Neural Networks often achieve high predictive performance but provide limited insight into how predictions are made. To improve transparency, this project incorporates two complementary explainability techniques: SHAP and LIME.

### SHAP (Global Interpretability)

SHAP is used to analyze the contribution of learned graph representations to model predictions. By assigning contribution scores to embedding features, SHAP provides a global view of model behavior across the dataset.

The SHAP analysis enables:

* Identification of influential learned feature dimensions.
* Feature importance ranking using SHAP values.
* Visualization of global prediction trends through summary and bar plots.
* Better understanding of overall model decision-making behavior.

### LIME (Local Interpretability)

LIME is used to explain individual molecular predictions by approximating the model locally around a selected instance. This provides insight into which learned feature representations have the greatest influence on a specific prediction.

The LIME analysis enables:

* Instance-level explanations for individual molecules.
* Identification of features contributing positively or negatively to a prediction.
* Local interpretation of model decisions.
* Improved transparency for specific prediction outcomes.

### Dual Explainability Approach

SHAP and LIME provide complementary perspectives on model behavior. SHAP offers a global understanding of feature influence across the dataset, while LIME focuses on local explanations for individual predictions. Together, they improve the interpretability of the proposed GIN framework and provide greater transparency in graph-based molecular property prediction.

Outputs include:

* Feature Contribution Analysis
* Instance-Level Explanations
* Local Prediction Interpretation
